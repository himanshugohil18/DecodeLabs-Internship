### Objective

Understand the fundamentals of Continuous Integration and Continuous Deployment.

### Tasks Performed

✅ Created CI/CD Workflow
✅ Configured GitHub Actions
✅ Automated Build Process
✅ Workflow Execution via Git Push

### Technologies Used

* GitHub Actions
* YAML
* Git Automation

### Sample Workflow

```yaml
name: CI Pipeline

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Run Pipeline
        run: echo "CI/CD Pipeline Running Successfully!"
```

### Skills Gained

* CI/CD Concepts
* Pipeline Automation
* Workflow Management
* DevOps Basics
